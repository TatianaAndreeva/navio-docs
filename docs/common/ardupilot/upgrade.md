## Overview

We use a conventional Debian scheme to upgrade software, i.e. using **APT**.

!!! warning
    Please, backup your parameters before proceeding!

## Upgrade

```bash
pi@navio: ~ $ sudo apt-get update && sudo apt-get dist-upgrade
```
